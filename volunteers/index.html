Developers

Help update and maintain freebassel.org by forking the Github repository at https://github.com/freebassel/freebassel.org and sending us pull requests.

Technologies
  • The website uses Ruby, JavaScript, HTML, CSS, Markdown, Jekyll, Foundation, and Inkscape.
  • You need a Github account to participate, and should be familiar with forking repositories and sending, discussing, and fixing pull requests.
  • You need a Google account for some tasks.
Tasks
  • Monitor news about Bassel Khartabil and add relevant web articles to the site.
  • Keep the supporters list up to date (requires a Google account).
  • Check Github issues to learn what else needs to be worked on, or email us for specific ideas.
